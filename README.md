## Start database

use the following command create, migrate and start the database:

* `npx prisma migrate dev --name init`

* `npx prisma db push`

* `npx prisma db seed`

## Pages endpoints

* Home page is at:
`localhost:3000`

* The admin page is at:
`localhost:3000/admin/jobs`

* The seeker page is at:
`localhost:3000/seeker/jobs`
